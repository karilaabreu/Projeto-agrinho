#  🌱 EcoAgro - Tecnologia Sustentável no Campo

O **EcoAgro** é um projeto educativo desenvolvido para o 
Programa Agrinho 2026 com o objetivo de mostrar como a tecnologia 
pode contribuir para a sustentabilidade no campo.

O projeto apresenta práticas sustentáveis, como:
- economia de água;
- reciclagem;
- uso de energia limpa;
- preservação ambiental;
- agricultura consciente.

A proposta é conscientizar estudantes e comunidades sobre a importância
do equilíbrio entre produção agrícola e meio ambiente.

---

## 🎯 Objetivo

Desenvolver um site interativo que incentive práticas sustentáveis
no meio rural utilizando tecnologia e educação digital.

---
